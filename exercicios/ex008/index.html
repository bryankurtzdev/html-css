Codigo-fonte

O comomando document.getElementById ('teste') e escrito em linguagem JavaScript.


Para deixarmos algum codigo de JS ou outra linguagem 'bonito' visualmente no nosso Html usamos primeiro a tag < pre > e dentro desta tag 'pre' nos colocamos a tag < code > e agora dentro da tag code nos escrevemos o nosso codigo, Abaixo ira ter uma demonstracao com as tags que mostrei acima e uma sem o uso de nenuma tag.

Escreva como esta abaixo: 


        
            < pre >
                 < code >
            
                 //Escreva o codigo
                 < /code >
            < /pre >

exemplos

codigo sem o uso de tags

// Função para gerar um número aleatório entre 1 e 10 function generateRandomNumber() { return Math.floor(Math.random() * 10) + 1; } // Variável para armazenar o número aleatório var randomNumber = generateRandomNumber(); // Função para verificar o palpite do jogador function checkGuess() { var guess = document.getElementById("guessInput").value; if (guess == randomNumber) { document.getElementById("result").innerHTML = "Parabéns! Você acertou!"; } else { document.getElementById("result").innerHTML = "Desculpe, tente novamente."; } }

codigo com o uso das tags 

        
            // Função para gerar um número aleatório entre 1 e 10
        function generateRandomNumber() {
            return Math.floor(Math.random() * 10) + 1;
        }

        // Variável para armazenar o número aleatório
        var randomNumber = generateRandomNumber();

        // Função para verificar o palpite do jogador
        function checkGuess() {
            var guess = document.getElementById("guessInput").value;

            if (guess == randomNumber) {
                document.getElementById("result").innerHTML = "Parabéns! Você acertou!";
            } else {
                document.getElementById("result").innerHTML = "Desculpe, tente novamente.";
            }
        }

percebeu que ao usar as tags no codigo ele ficou mais facil de entender e visualmente mais bonito e organizado?


Citacoes Silpes (sem quebra)

Para fazermos uma Citacao Simples usamos a tag < q >, Exemplo: Como diria o Pai de um amigo, o computador e um burro muito rapido

Citacoes Completas

Para fazermos um sitacao completa usamos a tag < blockquote >, Exemplo:

Ser feliz é uma responsabilidade muito grande. Pouca gente tem coragem. Nunca podemos julgar a vida dos outros, porque cada um sabe de sua própria dor e renúncia. Pretender-se que a vida dos homens seja sempre dirigida pela razão é destruir toda a possibilidade de vida.

Abreviacoes

Para mostrarmos em nosso HTML oque significa uma sigla usamos a tag < abbr >, Exemplo: Hj eu estudei Matematica. e assim quando passarmos o cursor em cima dessa abreviacao ira mostrar o significado dela.
