negrito / destaque

enfase / italico

nesta frase temos um termo em italico usando a tag < i > (nao semantica)

nestanfrase temos um termo em enfase usando a tag < em > (semantica)

texto marcado

podemos criar tambem um texto marcado usando a tag < nark >

texto e pequeno

Estamos criando um texto com letras pequenas usando a tag < small >.

texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido mas nao considerardo, usando a tag < del >

Podemos marcar um texto como inserido para dar uma enfase e indicar que ele foi adicionado depois, usando tag < ins >

texto sobrescrito

texto subscrito
