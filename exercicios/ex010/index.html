                 < a href="https://youtu.be/JlE0pzESf5g?si=mGNo8M3qFIZnMzP8"> Assista este Video para entender melhor as ancoras </a >

Agora fique com um breve explicacao das targets que podemos usar no href:

Para poder controlar onde o site de destino vai abrir, podemos usar o atributo target, que suporta os seguintes valores:

  • _blank vai abrir o link em uma nova janela em branco

  • _self vai abrir o link na janela ou frame atual padrão

  • _top vai desfazer todos os frames e abrir o destino no navegador completo

  • _parent similar ao uso do _top em uma referência à janela mãe
  • nome-do-frame caso esteja usando frames, indicar o nome da janela a abrir

Como o uso de frames é uma técnica quase em desuso, vamos nos basear apenas nas duas primeiras opções _blank e _self.

Links Externoss

Os links externos sao links que levam a outro site ou alguma outra coisa fora do nosso html.

O codigo para os links externos e a mesma coisa dos links internos. 

        
            Canal do Curso em Video

Em links externos utilizamos a mesma tag, mas mudamos o target dela, essa target diferente dos links internos que abrirao na mesma aba, a target dos links externos abrirao em uma nova aba.

para isso ocorrer precisamos adicionar algumas palavrinhas apos a URL dao link, essas palavrinhas sao: target="_blank" , esse comando ira fazer o link abrir em uma nova aba.

Tipos De Targets

  • _blank vai abrir o link em uma nova janela em branco
  • _self vai abrir o link na janela ou frame atual (padrão)
  • _top vai desfazer todos os frames e abrir o destino no navegador completo
  • _parent similar ao uso do _top em uma referência à janela mãe
  • nome-do-frame caso esteja usando frames, indicar o nome da janela a abrir
