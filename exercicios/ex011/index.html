codigo para imagens flexiveis.

o codigo para conseguirmos imagens flexiveis e: 

     
        < picture >
             < source media="(max-width: 750px)" srcset="imagens/fotopeq.png" type="image/png" >
             < source media="(max-width: 1050px)" srcset="imagens/fotomed.png" type="image/png" >
             < img src="Imagens/fotogrand.png" alt="Imagem flexivel" >
        
         < /picture >

esse codigo deve comecar com a imagem principal e acima dela colocar as de tamanhos menores, caso contrario o codigo nao funcionara. Nao precisamos necessariamente usar so tres tamanhos de imagens, isso vai de cada um.

Reproduzindo Audio

vamos aprender a reproduzir audios em HTML5


para adicionarmos um audio ao HTML5 usamos a tag < audio >, abaixo havera um exemplo! 


    
         < audio src="audios/happymistake.mp3" controls > Happymistake </audio >

e muito importante adicionar o "controls"!!

Dentro da tag & lt audio >, tambempodemos usar a tag < source >. Assim el


O Source funciona como o exemplo abaixo. 


    
         < audio preload="metadata" controls>
            < source src="audios/guanacast.mp3" type="audio/mpeg">
            < source src="audios/happymistake.mp3" type="audio/mpeg">
            < source src="audios/happymistake.wav" type="audio/wav">
            < source src="audios/happymistake.ogg" type="audio/ogg">
        </audio>

podemos usar todos esses tipos de audios em nosso site,porem e recomendavel excluir o audio .wav pois eles tendem a serem muito grandes e pesados assim deixando o site lento!!
